## 1. Análisis del Mercado de Códigos QR y Contactos Digitales

### Panorama del Mercado Global
El mercado de códigos QR generó $11.77 mil millones en 2022 y se proyecta que alcance $34.55 mil millones para 2028 (CAGR 19.6%). El crecimiento está impulsado por:
- Adopción masiva post-pandemia
- Pagos sin contacto
- Marketing digital
- Tarjetas de visita digitales

### Competidores Principales

| Competidor | Tipo | Fortalezas | Debilidades | Precio |
|------------|------|-----------|-------------|--------|
| **QRCode Monkey** | Generador QR gratuito | Popular, gratuito, muchas opciones | Genérico, sin enfoque en contacto | Gratis / $5-15 premium |
| **Canva** | Diseño + QR | Diseño hermoso, integración | Caro, complejo para beginners | $13-30/mes |
| **Haystack** | Tarjetas digitales | Bonitas, profesional | Requiere app, USA-centric | $8-25/mes |
| **Popl** | Tarjetas digitales | Sistema QR físico | Solo USA, costoso | $10-30/mes |
| **Linktree** | Links múltiples | Popular, familiar | NO es contacto, solo enlaces | Gratis / $5/mes |
| **Beaconstac (Uniqode)** | QR Enterprise | Robusto, analytics | Enfocado enterprise, caro | $25+/mes |

## 4. Logotipo

### Concepto de Logo
Un código QR que forma la silueta de un corazón o marca de verificación, representando "contacto guardado" + "éxito".

### Descripción Visual
```
+---------------------------+
|  +-------+   +-------+   |
|  | █ █ █ |   | ✓    |   |
|  | █ █ █ | → |      |   |
|  | █ █ █ |   +-------+   |
|  +-------+              |
+---------------------------+
   QR Code    →   Contacto
   (símbolo)     (verificación)
```

### Especificaciones del Logo
- **Forma**: Código QR estilizado con check
- **Proporción**: 1:1 (cuadrado)
- **Espacio de seguridad**: 20% del ancho
- **Tamaño mínimo**: 32x32px

### Versiones del Logo

| Versión | Uso | Fondo |
|---------|-----|-------|
| Completo (horizontal) | Website, documentos | Transparente |
| Icono solo | Favicon, redes, stickers | Transparente/Colores |
| Blanco | Videos, fondos oscuros | #1A1A1A |
| Negro | Impresión | #FFFFFF |

### Para el Diseñador
Al crear el logo, considera:
1. El QR debe ser reconocible instantáneamente
2. El check debe integrarse naturalmente (no parezca pegado)
3. Debe funcionar en tamaños pequeños (favicon)
4. Los colores deben coincidir con la paleta de marca

## 6. Tipografía

### Familia Principal
**Montserrat** (Google Fonts)
- Disponible gratuitamente
- Variantes: Thin, ExtraLight, Light, Regular, Medium, SemiBold, Bold, ExtraBold
- Unicode completo (acentos, ñ, etc.)

### Familia Alternativa
**Space Grotesk** (Google Fonts)
- Para títulos y énfasis
- Moderna y distintiva

### Escala Tipográfica

| Nivel | Elemento | Tamaño | Peso | Line-height |
|-------|----------|--------|------|-------------|
| H1 | Título principal | 48px | Bold (700) | 1.2 |
| H2 | Subtítulos | 36px | Bold (700) | 1.25 |
| H3 | Títulos sección | 28px | SemiBold (600) | 1.3 |
| H4 | Títulos subsection | 24px | SemiBold (600) | 1.35 |
| Body | Párrafos | 16px | Regular (400) | 1.6 |
| Body large | Párrafos destacados | 18px | Regular (400) | 1.6 |
| Small | Texto secundario | 14px | Regular (400) | 1.5 |
| Caption | Etiquetas, fechas | 12px | Regular (400) | 1.4 |
| Button | Texto botones | 16px | SemiBold (600) | 1 |

### Especificaciones CSS

```css
@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&family=Space+Grotesk:wght@400;500;600;700&display=swap');

:root {
  --font-primary: 'Montserrat', sans-serif;
  --font-display: 'Space Grotesk', sans-serif;
  
  --color-primary: #00C853;
  --color-primary-dark: #00A844;
  --color-text: #1A1A1A;
  --color-background: #FFFFFF;
  --color-background-alt: #F5F5F5;
  --color-accent: #2196F3;
  --color-warning: #FF9800;
  --color-error: #F44336;
}
```

## 8. Elementos Gráficos

### Iconografía
- Estilo: Línea simple, 2px stroke
- Redondeo: 4px en esquinas
- Tamaño: Múltiplos de 24px
- Color: Negro #1A1A1A o Verde #00C853

### Fotografías
- **Personas**: Profesionales en su ambiente de trabajo
- **Evitar**: Fotos de stock genéricas, personas sonriendo artificialmente
- **Preferir**: Capturas de pantalla del producto, demos reales
- **Estilo**: Natural, iluminación suave, colores reales

### Ilustraciones
- Estilo: Minimalista, línea gruesa
- Colores: Solo paleta de marca
- Uso: Decorativo, no explicativo
