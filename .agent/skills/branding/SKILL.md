# ActivaQR Brand Identity (V5 — Feb 2026)

## Brand Palette

| Token | Hex | Usage |
|---|---|---|
| `--primary` | `#f66739` | **Tomato/Orange.** Primary brand color used for everything: CTAs, accents, highlights, and borders. |
| `--primary-dark` | `#e5562d` | Darker shade of primary for hover states. |
| `--navy` | `#001549` | Primary text, headers, dark backgrounds. |
| `--sky` | `#68d3fe` | Light blue accent from the logo. |
| `--royal` | `#05509c` | Deep blue from the logo. |
| `--cream` | `#FCF9F5` | Page background. |

## Branding Policy

> [!IMPORTANT]
> As of Feb 2026, the **Green CTA color has been discarded** to maintain a unified tomato-based identity. All action buttons (CTAs) and secondary accents must use the primary tomato color (`#f66739`) with white text for maximum contrast.

## Typography & Title Rules

- **"QR" Highlight**: In blog titles, the letters "QR" are dynamically highlighted in `text-primary` (tomato).
- **Heading System**: Black/Bold weights with tight tracking (`tracking-tighter`).
- **Signature Style**: Italicizing key outcome words (e.g., *y no te olviden*).

## Logo Usage

- **Header**: `public/images/logo_header.png` — Horizontal format.
- **Square/Avatar**: `public/images/logo.png`.
