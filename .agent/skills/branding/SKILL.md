# ActivaQR Brand Identity (V6 — Feb 2026)

## Brand Palette

| Token | Hex | Usage |
|---|---|---|
| `--primary` | `#f66739` | **Tomato/Orange.** Base brand color for CTAs, main accents, and brand identity. |
| `--primary-dark` | `#e5562d` | Darker shade for hover states. |
| `--navy` | `#001549` | Primary text, headers, and dark background sections. |
| `--green` | `#66bf19` | **Success Accent.** Used for high-impact micro-highlights (e.g., "Escaneas el QR"). |
| `--sky` | `#68d3fe` | Light blue accent from the logo. |
| `--royal` | `#05509c` | Deep blue from the logo. |
| `--cream` | `#FCF9F5` | Page background. |

## Branding & Conversion Policy

> [!IMPORTANT]
> **Primary Identity**: All main buttons and structural CTAs must use Tomato (`#f66739`) with white text.
> **Conversion Accents**: Use Green (`#66bf19`) background highlights ONLY for high-conversion phrases or status indicators (e.g., "ACTIVADO").

## UI & Scaling Rules

- **Fixed Parallax**: Separator sections use `bg-fixed` background images with 100% opacity (no overlays) for a "window" effect.
- **Pricing Pillars**: Always present 3 options (Rule of 3). The middle plan must be the "anchor" with higher visual elevation and a "Más Popular" badge.
- **Micro-Highlighting**: Highlight "QR" in the branding (ActivaQR) or technology mentions using green backgrounds or tomato text depending on the context.
