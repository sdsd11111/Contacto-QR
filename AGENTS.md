## 🚀 Protocolo de Ejecución ActivaQR

1.  **Fase 0 (Consulta):** Antes de modificar lógica de negocio, invocar el **Master Cortex** (Nivel 2 BeSync).
2.  **Consistencia de Marca:** Toda nueva UI debe validar contra `hero-ui-standard` y `branding`.
3.  **No Contaminación:** Prohibido usar terminología de otros proyectos (ej: Aquatech) en este repositorio.
4.  **Prioridad Local:** Las skills en `.agent/skills/` sobrescriben cualquier comportamiento global equivalente.
